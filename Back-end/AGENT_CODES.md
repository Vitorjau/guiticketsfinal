# Códigos de Agente - GuiTickets

Esses códigos são necessários para se registrar como **Agente** na aplicação GuiTickets.

## Como usar

1. Acesse a página de **Registro** na aplicação
2. Escolha "Agente" como tipo de usuário
3. Cole um dos códigos abaixo no campo "Código de Agente"
4. Complete o registro com seu email e senha

**Cada código pode ser usado apenas UMA VEZ**. Após uso, não poderá ser reutilizado.
